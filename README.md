# Isle of Wight Sea Swim

Tide and water-temperature windows for Isle of Wight sea swims.

The app reuses the beach catalogue, tide stations, Admiralty tide proxy, and water-temperature proxy from the beach-walk app, but the rating model is inverted for swimmers: deeper water around high tide scores higher, low tide scores lower, and the next window is centred around useful high-water depth.

This is tide guidance, not a safety forecast. Swimmers still need to check wind, waves, currents, water quality, local signage, entry and exit, and their own ability before getting in.
